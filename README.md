# Modeling self organization for WSN

**Running app**

```git clone https://github.com/Oleg-Ovcharenko/Diploma.git```

```cd Diploma```

**Installation (development)**

```npm run build:dev```

After running app need open in browser: http://localhost:8080/

**Installation (propduction)**

```npm run build:prod```
